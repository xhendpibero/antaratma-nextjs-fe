## Bootstrap 4

Design System NextJS is built on top of the much awaited Bootstrap 4 (Reactstrap). This makes starting a new project very simple. 

## Quick start

- Clone the repo: `git clone https://github.com/xhendpibero/antaratma-nextjs-fe/`.
- `npm install or yarn`
- `yarn dev`
